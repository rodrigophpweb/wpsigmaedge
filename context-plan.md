# O Projeto

A Sigma Edge é uma empresa que vende produtos e presta serviços na area de calibração de instrumentos. Objetivo desse site é apresentar os produtos, serviços e informações institucionais da empresa. 

O relacionamento entre empresa e clientes se dará contato via WhatsApp quando uma pessoa clicar sobre o botão cotar dentro das singles seja de produto ou serviço.
--- 

## Tecnologia
O Projeto será desenvolvido tecnologias open source como:
- PHP
- WordPress
- MariaDB
- HTMX
- JavaScript (Vanilla)

## Componentes do Projeto.
Abaixo vamos listar os nomes dos componentes onde praticamente será repetido no projeto.

- Header : Composto por Logotipo, menu do topo, barra de busca, botão solicite um orçamento. (Será exibido em todas páginas do projeto.)
- Footer : O componente é composto por logotipo, barra de navegação de redes sociais, Barra de Menu com 3 colunas, formulário com campo de input email para cadastrado de newsletter.
- Section Map: Teremos de background da seção um google maps, no canto esquerdo na parte superior um box com o logotipo e o Texto Sigma Edge | Matriz São Bernado.
  - No lado direito teremos um formulário de contato com:
    - h3 {Fale com a gente}
    - p {Para dúvidas sobre os nossos serviços, produtos, cotações, ou outros assuntos, basta preencher os campos a seguir que a nossa equipe manterá contato}
    - Input para Nome
    - Input para email
    - Input para Celular
    - Input para Assunto
    - Input Textarea para mensagem
    - Botão solicitar contato

- Section Blog: (Essa seção vai retornar os últimos 03 posts do site com as seguintes informações):
  - h2 {Últimas postagem do Blog}
  - Categoria do Post
  - h3 {Título do Post}
  - p {Trazer o resumo do the_contetn()}
  - a {Link para o post com o texto saiba mais}
